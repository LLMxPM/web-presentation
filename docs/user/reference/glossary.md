# 术语表

| 术语 | 说明 |
| :--- | :--- |
| Backend | 平台控制面，负责用户、权限、数据、AI Agent、预览上下文、构建任务和产物托管 |
| Editor | 创作工作台，负责工作空间、项目、页面、组件、资源、主题、样式和 AI 侧边栏界面 |
| Runtime | 预览与构建执行引擎，负责页面渲染、组件预览、截图、诊断和构建 |
| 工作空间 | 团队资产和权限隔离单位 |
| 项目 | 一组页面、主题样式和构建配置的集合 |
| 页面 | 最终被预览和构建的内容单元 |
| 资源 | 图片、图标、视频、字体、DrawIO、Mermaid、图表、公式等素材 |
| 组件 | 工作空间级可复用 Vue 组件 |
| 主题 | 品牌色、Logo、页面图标和字体绑定等视觉基础配置 |
| 样式库 | 页面展示配置和 Markdown 样式规范的集合 |
| Runtime Kit | Runtime 对页面和组件公开的版本化能力集合 |
| previewSchema | 组件预览输入配置，用于验证组件典型状态 |
| build snapshot | Backend 为构建任务固化的一次性项目快照 |
| HITL | Human-in-the-loop，指需要用户确认的 AI 工具调用流程 |
