# 当前限制

本文档记录用户侧需要知道的已落地能力、建设中事项和使用边界。更细的开发路线应以项目 issue、PR 和开发文档为准。

## 已落地能力

- 登录、多用户隔离、工作空间、项目和页面管理。
- 资源库、组件库、主题库、样式库和字体注册。
- AI Agent 会话、上下文注入、工具确认和会话恢复。
- 页面预览、组件预览、截图、构建任务和构建产物托管。
- 单机 compose 部署和 Runtime 子模块接入。

## 建设中事项

- 跨项目资产治理和使用关系运营视图。
- Dashboard 和更完整的项目状态分析。
- Runtime 反向回传和更细粒度诊断。
- 更完整的模板包、案例和迁移工具。

## 使用边界

- 平台更适合长期项目和团队资产沉淀，不是一次性生成单个 PPT 文件的最短路径。
- AI 输出需要通过预览和人工确认，不应直接视为最终交付。
- 生产环境部署前必须配置密钥、数据库、对象存储、Runtime 地址和备份策略。
