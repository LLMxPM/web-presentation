# AI 协作创作指南

AI 协作围绕当前用户、工作空间、项目、页面和资产上下文展开。平台会把必要上下文提供给 Agent，并通过工具确认机制控制高风险操作。

## 当前助手

平台围绕不同创作对象提供助手能力，包括页面内容助手、组件助手和资源助手。不同助手看到的上下文范围不同，能调用的工具也不同。

## 使用原则

- 先描述目标，再给素材和限制。
- 修改已有页面时说明范围，避免让 AI 猜测。
- 工具确认前检查对象、参数和影响范围。
- 对视觉结果不满意时，给出具体反馈，例如“减少装饰元素”“提高标题层级”“压缩信息密度”。
