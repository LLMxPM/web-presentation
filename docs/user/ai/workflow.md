# 协作流程与提示词

## 适合交给 AI 的任务

- 根据主题生成页面大纲或单页初稿。
- 把长文本压缩成演示文案。
- 根据现有组件生成页面源码。
- 调整页面布局、视觉层级、配色和字体密度。
- 解释资源、组件或样式如何在当前页面中使用。

## 提示词建议

有效提示词通常包含目标、受众、素材、风格、限制和期望输出。例如：

```text
请为“季度经营复盘”生成一页核心结论页，面向管理层。
使用当前项目主题，信息密度偏高但要保持可读。
必须包含三个结论、一个风险提示和一个下一步行动。
优先使用工作空间里已有的数据卡片组件。
```

## 协作节奏

先让 AI 生成结构，再逐步调整内容和样式。不要一次性要求 AI 同时完成选题、文案、布局、组件、资源、构建和导出；拆成多轮通常更稳定。

## 请求示例

- “根据当前页面内容，把标题改得更像汇报型结论。”
- “用已发布的三列指标卡组件重排这一页。”
- “检查这页是否引用了不存在的资源。”
- “保留文案含义，把视觉风格调整为更克制的企业报告页。”
