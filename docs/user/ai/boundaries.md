# 能力边界

## 内容助手

内容助手主要处理页面结构、文案、样式和资源引用。它适合生成页面初稿、改写内容、解释页面代码和调整视觉层级，但不应直接承担生产运维、权限配置或密钥处理。

## 组件助手

组件助手围绕工作空间组件工作，适合生成组件草稿、补充参数说明、调整 previewSchema 和解释引用方式。发布组件前仍需要人工确认源码、依赖和预览结果。

## 资源助手

资源助手适合帮助理解资源用途、生成资源说明、建议引用素材和检查资源依赖。资源删除、替换和批量归档应由用户确认影响范围后执行。

## 可见性边界

AI 只能基于当前用户有权访问的工作空间、项目、页面、组件和资源工作。工具确认并不等于权限放开，所有写入仍应经过 Backend 权限校验。
