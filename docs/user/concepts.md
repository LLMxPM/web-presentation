# 核心概念

`web-presentation` 把演示内容拆成平台对象，并围绕这些对象提供资产复用、AI 上下文注入、预览、截图和构建能力。

## 工作空间

工作空间是团队资产和权限隔离的基本单位。用户在工作空间内管理项目、资源、组件、主题、样式和字体，AI 也会以当前工作空间为边界读取可见上下文。

## 项目

项目是一组演示页面和配置的集合，通常对应一个 PPT、专题报告、数据解读页或图文卡片集合。项目会绑定页面比例、主题、样式、资源引用和构建配置。

## 页面

页面是最终被预览和构建的内容单元。页面源码通常以 Vue 页面能力表达，既可以直接引用工作空间资源，也可以引用已发布的工作空间组件和 Runtime Kit 能力。

## 资源

资源是页面、组件、主题和字体注册可以引用的素材，包括图片、图标、视频、字体、DrawIO、Mermaid、图表、公式和可编辑内容资源。资源管理流程见 [资源管理](./workflows/resources.md)。

## 组件

组件是工作空间级可复用 Vue 组件，适合沉淀封面、目录、数据卡片、图表区块、报告模板等稳定结构。组件采用草稿与发布版本模型，使用流程见 [组件管理](./workflows/components.md)。

## 主题、字体与样式

主题负责品牌色、Logo、页面图标、字体绑定等视觉约束；字体注册负责把字体资源转为页面可引用声明；样式库用于沉淀页面展示配置和 Markdown 样式规范。使用流程见 [主题、字体与样式](./workflows/design-system.md)。

## AI 会话

AI 会话围绕当前页面、项目、组件或资源展开。Backend 会按范围注入必要上下文，并通过工具确认机制把高风险操作交给用户确认。详见 [AI 协作创作](./ai/README.md)。

## Runtime

Runtime 是预览与构建执行引擎。Editor 通过 iframe 展示 Runtime 渲染结果，Backend 负责签发预览上下文、保存构建任务和托管产物。用户侧流程见 [预览、截图与构建](./workflows/preview-build-export.md)。
