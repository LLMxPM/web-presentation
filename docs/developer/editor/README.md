# Editor 开发文档

Editor 是面向用户的创作工作台，负责登录、工作空间、项目、页面、组件、资源、主题、样式、AI 侧边栏、账户 AI 设置、预览 iframe 和构建入口。

## 文档导航

| 文档 | 内容 |
| :--- | :--- |
| [前端结构约定](./structure.md) | 视图、组件、组合式逻辑、状态和 API 层拆分 |
| [AI 侧边栏](./ai-sidebar.md) | 工具说明展示、会话恢复、HITL 确认和上下文 |
| [Editor 测试](./testing.md) | 类型检查、Vitest 和前端质量门禁 |

## 修改原则

- 新增能力时优先复用已有 UI 组件和交互模式。
- API 请求层不要散落在视图组件里。
- 账户 AI 设置页要展示完整工具说明和只读契约信息。
- 预览 iframe 只呈现 Runtime 结果，不承载 Runtime 内部逻辑。
